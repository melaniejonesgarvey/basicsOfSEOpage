The following are terms commonly utilized in reference to SEO campaigns. Familiarize yourself with these terms when getting to know the world of SEO.

Search Engine Marketing
Search Engine Marketing (SEM) is frequently confused with SEO. The most basic way to separate the two is to keep in mind SEM is paid while SEO is organic. Listed on this page is a screen shot of a Search Engine Results Page (SERP). The first result is labeled as an ad which is a paid SEM position. The following two results are generated by SEO.
Crawlers
A program used by search engines that systematically examines the internet for the most up to date information. Also known as a spider, robot or simply bot.
Index
The database created by crawlers for search engines. If a page is not “indexing” well that means that search engines are not finding it which also means it is difficult for users to find.
SERP
Search Engine Results Page – This is the page on which all findings from a search engine inquiry are listed.
White Hat SEO
Strategies that focus on increasing quality of the website instead of how to outsmart the search engine for better results.
Black Hat SEO
A practice that is based around only gaining preferred results from search engines while actively going against their policies.
On-Page SEO
On-Page SEO is a technique that uses elements of the website such as content and code to increase SEO performance.
Off-Page SEO
Method that utilizes external resources for better positioning on the SERP. Please see below for examples of external resources.
  • Social Media
  • Online talk such as reviews, postings and forums.
  • Linking
PageRank
A term from SEO’s history since Google removed PageRanks from public view in spring 2016. Previously this algorithm assigns a score to a website and then lists it in ranking order on the SERP.
